# Networking Basics

## What is networking
Networking is the practice of connecting devices so they
can communicate with each other.

## Key concepts
- IP address
- MAC address
- DNS
- Ports
- Protocols

## Common protocols
- TCP
- UDP
- HTTP / HTTPS
- FTP
- SSH

## Why networking matters in cybersecurity
Most cyber attacks happen over networks.
Understanding networking helps identify suspicious
traffic, misconfigurations and attacks.
